# Docker compose with multiple local containers

To build:

```sh
docker build -t milhamhidayat/visits:latest .
```

To run:

```sh
docker run milhamhidayat/visits:latest
```

Run docker redis

Use docker compose:

```sh
docker-compose up
```

Run docker compose in background:

```sh
docker-compose up -d
```

Stop containers

```sh
docker-compose down
```
